Priority ODATA — API Reference

Base URL:   https://YOUR_PRIORITY_HOST/odata/Priority/tabula.ini/YOUR_COMPANY/
Auth:       Basic (PRIORITY_USER : PRIORITY_PASSWORD) — both live in .env

FIND CUSTOMER (sheet column C → CUSTNAME)

GET /CUSTOMERS?$select=CUSTNAME,CUSTDES

Sample response:
{
  "@odata.context": ".../$metadata#CUSTOMERS(CUSTNAME,CUSTDES)",
  "value": [
    { "CUSTNAME": "CUST001", "CUSTDES": "Example Customer A" },
    { "CUSTNAME": "CUST002", "CUSTDES": "Example Customer B" }
  ]
}

FIND WAREHOUSE BY DESCRIPTION (sheet column A branch → TOWARHSNAME)

GET /ZANA_WARHSDES_EXT_FL

Sample response:
{
  "@odata.context": ".../$metadata#ZANA_WARHSDES_EXT_FL",
  "value": [
    { "WARHSNAME": "1", "WARHSDES": "Example Branch Name", "WARHS": 1 }
  ]
}

FIND PRODUCT BY DESCRIPTION (sheet column G → PARTNAME)

GET /ZANA_PARTDES_EXT_FLA

Sample response:
{
  "@odata.context": ".../$metadata#ZANA_PARTDES_EXT_FLA",
  "value": [
    { "PARTNAME": "1000001", "PARTDES": "Example Product", "PART": 1 }
  ]
}
